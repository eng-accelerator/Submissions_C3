# Installation Instructions:
# 1. Create virtual environment: python -m venv venv
# 2. Activate virtual environment:
#    - Windows: venv\Scripts\activate
#    - Linux/Mac: source venv/bin/activate
# 3. Install dependencies: pip install -r requirements.txt
#
# Alternative Installation (faster):
# pip install --upgrade pip
# pip install -r requirements.txt --no-cache-dir
#
# For development:
# pip install -r requirements.txt -e .
#
# System Requirements:
# - Python 3.9 or higher
# - PostgreSQL 12 or higher
# - Minimum 4GB RAM
# - Minimum 2GB disk space for dependencies
#
# Optional Dependencies:
# - For GPU support: pip install faiss-gpu (instead of faiss-cpu)
# - For advanced NLP: python -m spacy download en_core_web_sm
# - For NLTK data: python -m nltk.downloader all
#
# Environment Variables Required (create .env file):
# - DATABASE_URL=postgresql+psycopg2://user:pass@host:port/dbname
# - OPENROUTER_API_KEY=your_openrouter_key
# - OPENAI_API_KEY=your_openai_key
# - SECRET_KEY=your_secret_key
#
# Platform-Specific Notes:
# - Windows: Some packages may require Visual C++ Build Tools
# - Mac M1/M2: Use architecture-specific builds where needed
# - Linux: May need additional system packages (libpq-dev, python3-dev)
#
# Troubleshooting:
# - If psycopg2 fails, try: pip install psycopg2-binary
# - If lxml fails on Windows, download prebuilt wheel from unofficial binaries
# - Clear pip cache if needed: pip cache purge
#
